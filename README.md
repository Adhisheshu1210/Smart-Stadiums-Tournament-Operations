# üìñ Overview

**Smart Stadiums & Tournament Operations** is an AI-powered smart stadium platform built for the **FIFA World Cup 2026 Hackathon** under **Challenge 4**.

The application demonstrates how **Generative AI** can improve stadium operations and enhance the overall tournament experience for:

- üë®‚Äçüë©‚Äçüëß Fans
- ü¶∫ Volunteers
- üëÆ Stadium Staff
- üèü Venue Operators
- üéØ Event Organizers

Using **Google Gemini AI**, the platform delivers intelligent assistance, operational recommendations, stadium navigation guidance, multilingual support, accessibility enhancements, and simulated crowd intelligence.

The project has been designed as a lightweight MVP that is easy to deploy, secure, accessible, and optimized for hackathon evaluation.

# üì¶ Installation

## Prerequisites

Before running the project, ensure you have installed:

- Node.js 18+
- npm

Verify versions:

```bash
node -v

npm -v
```

Clone the repository:

```bash
git clone https://github.com/Adhisheshu1210/Smart-Stadiums-Tournament-Operations.git
```

Navigate into the project:

```bash
cd Smart-Stadiums-Tournament-Operations
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

You can obtain your API key from:

https://aistudio.google.com/app/apikey

Start the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
