# 📖 Overview

**Smart Stadiums & Tournament Operations** is an AI-powered web application developed for the **FIFA World Cup 2026 Hackathon – Challenge 4**.

The platform demonstrates how **Generative AI** can improve stadium operations, assist fans and volunteers, provide intelligent navigation guidance, and support organizers with operational insights.

The application combines a modern React interface with Google's Gemini AI to deliver interactive and intelligent assistance for smart stadium experiences.

---

# 🎯 Challenge Statement

Build a GenAI-enabled solution that enhances stadium operations and the tournament experience for fans, organizers, volunteers, and venue staff through intelligent assistance and operational decision support.

# 🚀 Getting Started

## Prerequisites

- Node.js 18+
- npm

---

## Clone Repository

```bash
git clone https://github.com/Adhisheshu1210/Smart-Stadiums-Tournament-Operations.git

cd Smart-Stadiums-Tournament-Operations
```

---

## Install Dependencies

```bash
npm install
```

---

## Environment Variables

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

You can obtain a Gemini API key from:

https://aistudio.google.com/app/apikey

---

## Run Development Server

```bash
npm run dev
```

Open your browser:

```
http://localhost:3000
```

---

## Run Tests

```bash
npm run test
```

---

## Lint TypeScript

```bash
npm run lint
```

---

## Build Production

```bash
npm run build
```

---

## Start Production Server

```bash
npm start
```
