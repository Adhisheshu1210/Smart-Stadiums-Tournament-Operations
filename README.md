# 📖 Overview

**StadiumMind AI** is a lightweight AI-powered Smart Stadium platform designed for the **FIFA World Cup 2026 Hackathon**.

The application enhances the match-day experience for fans, volunteers, organizers, and stadium staff using **Google Gemini AI**, intelligent recommendations, real-time dashboards, accessibility tools, and multilingual assistance.

# 🚀 Getting Started

## Prerequisites

- Node.js 18+
- npm

---

## Installation

Clone the repository

```bash
git clone https://github.com/Adhisheshu1210/Smart-Stadiums-Tournament-Operations.git
```

Go inside the project

```bash
cd Smart-Stadiums-Tournament-Operations
```

Install dependencies

```bash
npm install
```

---

# 🔑 Environment Variables

Create a file named

```text
.env.local
```

Add your Gemini API Key

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Get your API key from:

https://aistudio.google.com/app/apikey

---

# ▶️ Run Locally

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

# 📦 Build Production

```bash
npm run build
```

Start production server

```bash
npm start
```
